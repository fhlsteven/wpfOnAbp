### 介绍
基于Prism的WPF应用示例

#### 开发工具
  IDE: Visual Studio 2019 

#### 说明
项目默认下载后,已连接至测试服务器, 如果需要本地部署,  只需要配置数据库参数即可.
运行**Consumption.Api** 项目,即可生成对应数据库, 生成之前请检查配置文件。
如旧版本已生成, 请手动删除后启动API项目。
